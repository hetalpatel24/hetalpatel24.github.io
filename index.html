Summary: Latitude vs. X



The purpose of this project was to analyze how weather changes in different cities as we get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.


After assembling the dataset, I used Matplotlib to plot various aspect of the weather vs. Latitude. Factors we looked at included: temprature, cloudiness, windspeed and humidity. This site provides the source data and visulization created as a part of analysis, as well as explaination and descriptions of any trends and corelations witnesed.
